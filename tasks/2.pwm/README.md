# 任务三：PWM呼吸灯

## 📃 任务描述

在这个任务中，你需要编写一个最基本的代码，使用link烧录到最小系统板上，实现呼吸灯的效果

## 🎯 任务目标

使用 C 语言编写一个程序：实现板载led的明暗变化

ps：如果可以的话，使用按键来控制明暗变化的频率，如1s内**暗->亮->暗**变化1/2次
